# Flow-HQ Product Vision

Flow-HQ is a unified growth operating system.

Users move through:

Discover → Plan → Execute → Learn → Improve → Grow


Products:

1. Flow Dashboard
User command center.

2. Flow Explore
Discovery hub.

3. Flow Roadmaps
Helps users find structured paths.

4. Flow Projects
Turns ideas and plans into execution.

5. Flow Business AI
Provides business intelligence.

6. Flow Playlists
Provides structured learning.


Important:

These are NOT separate apps.

They are modules inside the Flow ecosystem.


Never create isolated experiences.
Every module should connect to another module.
