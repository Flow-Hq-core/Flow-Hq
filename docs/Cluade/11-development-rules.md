# Development Rules


Before creating anything:


1. Check existing module.


2. Reuse shared components.


3. Do not duplicate UI.


4. Do not move files without reason.


5. Preserve existing design.


6. Keep features isolated.


7. Update documentation when adding major features.



Forbidden:


Creating:

apps/new-product

random pages

duplicate dashboards

duplicate UI libraries



Correct:


New feature goes inside existing module.
