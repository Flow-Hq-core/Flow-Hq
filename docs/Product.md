Flow-HQ

A platform of AI-powered tools that help people build businesses, build products, and learn faster.

Each product has a very clear responsibility.

⸻

🗺️ Flow Roadmaps

Purpose

Discover structured roadmaps.

Think of it as the business knowledge library.

It answers:

“What should I learn or implement?”

It does NOT generate roadmaps (for now).

Users can

* Browse Business 101
* Browse Agency roadmap
* Browse Coffee Shop roadmap
* Browse Restaurant roadmap
* Browse Marketing roadmap
* Browse Sales roadmap

Each roadmap contains

* Topics
* Explanations
* Resources
* Links
* Related templates
* Related workflows

It is essentially a structured encyclopedia.

⸻

🤖 Flow Business AI

This is NOT a roadmap.

This is the business consultant.

It answers

“Help me solve this business problem.”

It can

* Chat
* Diagnose businesses
* Generate SOPs
* Generate workflows
* Generate templates
* Generate checklists
* Generate business canvases
* Brainstorm ideas
* Analyze strategies
* Find opportunities
* Review offers
* Improve marketing
* Improve pricing
* Build systems
* Generate documents

This is where all AI generation happens.

⸻

💻 Flow Project

Completely separate domain.

This is for

Software

Hardware

Engineering

Startup products

Things like

* Brainstorm
* User flows

⸻

▶️ Flow Playlist

Purpose

Learning.

User says

“I want to learn React.”

or

“I want to learn Marketing.”

Flow creates

A curated learning playlist.

Nothing more.

No roadmap.

No AI consultant.

Just the best learning path.
